## 配置

> 因为wallpaper engine 不支持文字输入，所以配置向导不能完全实现

点击桌面右上角自动配置向导可进行桌面背景和所在城市位置配置

config文件中可根据注释进行选择性的配置，如背景上的文字内容

## wallpaper engine 打开html文件

1. 打开wallpaper engine
2. 选择左下角 open from file 
3. 选择background.html
4. 打开即可
